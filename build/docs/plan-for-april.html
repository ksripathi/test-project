2 Design

2.1 Implemented and working APIs for Lab Data Service

The APIs for the lab data service have been implemented which enables us to search the labs with experiments,institutes,developers and discipline. The operations allowed for the admin user are adding labs, updating labs, searching labs etc which follows similarly for operations in experiments,institutes, developers and disciplines as well.

The following APIs are being implemented. We are expecting a release on 15th Feb, 2017 with the following entitites :

  • Lab
  • Experiment
  • Institute
  • Developer
  • Discipline
  • Asset - In progress
  • Hosting Information - In progress
  • Integration Status - In progress

6 Realization

Implementing the above plan will result into the following -

  • Labs and experiments as services. Github repository link for the same.
  • Able to search and get the updated info about any lab by experiments,institutes,developers etc.
  • The APIs could be served to other services for resource IDs and fetching information about labs.
  • Dynamic serving of the lab data using content server on any hosting platform like Open Edx, Moodle etc.
