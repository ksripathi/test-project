1 Object Delegate

Object Delegate encapsulates a set of functions where each function performs a specific task. With the use of these delegates, the core implementation of the system will not alter but by plugging delegates that are specific to each environment, for example, be it either objects or persistence, the system for that particular environment is derived while not modifying the system class.

1.1 Initialize Object Delegate

Provides all the delegates that operate on objects.

1.1.1 class_object_delegate

class ObjectDelegate():
    def __init__(self):
        self.role_set = []
        self.user_set = []
        self.active_user_set = []
        self.institute_set = []
        self.oc_set = []
        self.nc_set = []
        self.workshop_set = []
        self.artefact_set = []
        self.role_set = self.initialize_role_set()
        self.user_set = self.initialize_user_set()

1.1.2 initialize_role_set

    def initialize_role_set(self):
        Role_admin = Role(name="admin", centre_oc=None, centre_nc=None)
        Role_guest = Role(name="guest", centre_oc=None, centre_nc=None)
        Role_noc = Role(name="noc", centre_oc=None, centre_nc=None)
        Role_reviewer = Role(name="reviewer", centre_oc=None, centre_nc=None)                
        self.role_set = [Role.admin, Role.guest, Role.noc, Role.reviewer]
        return self.role_set

1.1.3 initialize_user_set

    def initialize_user_set(self):

        admin_user = User(name=Name(name=Config.admin_name),
