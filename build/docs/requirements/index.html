1 Introduction

This document describes all the requirements of lab-data-service.

2 Requirements

2.1 Req1

  • Ability to store & retrieve the data about labs based on lab and experiment specifications.

2.2 Req2

  • Ability to query the data about the lab information with all possible queries.

2.3 Req3

  • Ability to get resource ids of experiments and labs.

2.4 Req4

  • Footer need to be served from Lab Data Service. All the portals like OpenedX, Outreach Portal, and Feedback should be able to use the footer.

2.5 Req5

  • Ability to provide information about experiment, lab, institute and discipline Assets.

2.6 Req6

  • Ability to provide Hosting Information of lab and experiment.

2.7 Req7

  • Ability to provide the lab and experiment Integration status.

2.8 Req8

  • Lab Data Service should generate National Digital Library specification. Issue #45

2.9 Req9

  • Ability to search about labs using all possible 'keywords'. Issue #46

2.10 Req10

  • LDS need to capture the metadata for google crawling. Issue #47
